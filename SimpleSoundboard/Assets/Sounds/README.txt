Drop bundled example sounds here (.wav, .mp3, or .ogg).

Any audio file in this folder is automatically added to the soundboard the
first time the app launches after it appears here. Each file is seeded once:
- Remove it in the app and it won't come back.
- Add a new file here and it shows up on the next launch.

These ship with the app (copied next to the executable), so they're available
on a fresh install with no sounds configured yet.
